 # 📸 MemoAlbum

MemoAlbum is a digital wedding memory platform that allows photographers to create, manage, and share interactive wedding albums with couples. The system supports three roles: Admin, Photographer, and Couple.

---

## ✨ Features

### 👤 Admin
- Send email invitations to photographers
- Manage photographer accounts
- Monitor system usage
- Control user access and permissions

---

### 📷 Photographer Panel
- Create wedding albums
- Add album details:
  - Album Name
  - Wedding Date
- Set access control:
  - Public Gallery
  - Private Link
  - Password Protected
- Upload photos and videos (Drag & Drop)
- Storage limit: **5GB per album**
- Design albums using templates:
  - Book View
  - Scroll View
- Set cover photo
- Invite couples via email

---

### 💑 Couple Panel
- Login via email invitation link
- View wedding album securely
- Access photos and videos
- View wedding summary/details
- Payment confirmation (if applicable)
- Share album with others (based on permissions)

---

## 🔄 System Workflow

1. Admin invites photographer via email  
2. Photographer logs in and creates album  
3. Photographer uploads media and designs album  
4. Photographer invites couple users  
5. Couple logs in using invitation link  
6. Couple views and accesses final wedding album
